# Vivado Design Suite Tutorials

This directory contains visual step-by-step guides to help you navigate the Xilinx Vivado environment. These presentations are designed to get you from a blank project to a programmed Nexys A7 board.

---

## Tutorial Catalog

### 1. Vivado Introduction (Intro Guide)
**File:** `Tutorial vivado intro.pptx`
* **Focus:** Initial setup and project creation.
* **Topics Covered:**
    * Starting a new project.
    * Selecting the correct FPGA part (Nexys A7-100T / xc7a100tcsg324-1).
    * Importing or creating source files (VHDL/Verilog).
    * Defining constraints (.xdc).

### 2. Vivado Complete Workflow
**File:** `Tutorial vivado.pptx`
* **Focus:** The full design flow and hardware programming.
* **Topics Covered:**
    * Running RTL Analysis.
    * Synthesis and Implementation process.
    * Reviewing timing and utilization reports.
    * Generating the Bitstream file.
    * Using Hardware Manager to program the FPGA.

---

## Prerequisites
* **Software:** Vivado Design Suite (Standard Edition) installed.
* **Hardware:** Nexys A7 (Artix-7) board and Micro-USB cable.
* **Board Files:** Ensure the Digilent board files are installed in Vivado before starting.

---

## How to Use
1. Download the `.pptx` files to your local machine.
2. Follow the slides alongside your open Vivado instance.
3. For the source code mentioned in these guides, refer to the `/src` and `/examples` folders in this repository.
